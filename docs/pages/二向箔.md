# 二向箔

二向箔，一种可将三维空间降为二维平面的宇宙级武器，是《死神永生》中出现的终极毁灭手段。太阳系最终被二向箔攻击，整个星系（包括地球）被展平为一幅二维画卷。

## 原理

二向箔是一种在高维空间中展开的薄膜，一旦释放便开始将周围空间降维——将三维空间"压"成二维平面。降维过程以光速向外扩展，无法阻止。

被降维的空间内，所有三维物体都被展平，物理规律随之改变，生命无法存在。

## 使用

在[[黑暗森林法则]]体系下，二向箔是高等文明对威胁文明实施降维打击的标准手段：不留痕迹、不可逆、彻底。

当太阳系被某高等文明锁定为威胁后，一枚二向箔被投入太阳系，太阳系随后以光速扩展的方式被降为二维。[[程心]]与艾AA乘光速飞船逃出，成为极少数的幸存者。

## 在《死神永生》中的意义

二向箔事件标志着人类文明的实质性终结，也是刘慈欣对"黑暗森林"宇宙的终极展示——宇宙中存在比[[三体文明]]更高等、更冷酷的力量。
