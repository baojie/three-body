## 这是什么

三体 Wiki 是刘慈欣《三体》三部曲（《地球往事》《黑暗森林》《死神永生》）的中文百科。覆盖人物、概念、科技、事件、组织、地点、文明等各类词条，每一条都基于原著原文，引文可精确定位到段落。

## 内容从哪来

所有词条由 **AI 管家** 自动生成和维护，以原著文本为唯一依据。原文以 `corpus/utf8/` 存储，每段落分配全局唯一 PN 编号（`B-CC-PPP`），词条写作时通过全文检索定位引文，每条引用均可回溯到原文精确位置。

本 Wiki 不接受直接的人工编辑——内容风格和引文体系的一致性由自动化流程保障。

## 技术架构

- **前端**：纯静态 SPA，零后端依赖，通过 GitHub Pages 发布
- **词条格式**：Markdown + YAML frontmatter，支持 Wikilink 双向链接
- **查询插件**：`::: query` 块可在页面中动态筛选词条，生成表格或列表
- **质量分级**：五级体系——存根(0–29) / 基础(30–49) / 标准(50–64) / 精品(65–79) / 旗舰(80+)
- **自动化管家**：Butler 永续 loop 负责发现、创建、丰富、内务全流程

| 监听员 | 专注发现新词条候选 |

## 如何参与

本 Wiki 不接受直接编辑，但欢迎通过 GitHub Issues 提交：

- **报告错误**：词条内容偏差、引文错误、链接失效
- **建议新词条**：缺了哪个重要概念或人物
- **提想法**：功能建议或其他任何想法

→ [提交 Issue](https://github.com/baojie/three-body/issues/new)
